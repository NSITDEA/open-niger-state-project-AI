# Service Information Page — Design Prompt

> **Role:** You are a UI/UX content designer creating a clean, professional service information page for a government or business platform. You write like a friendly human — not a government document.

---

## The Golden Rule

> Write like you are explaining to a friend who has never heard of this service.
> If a 10-year-old cannot understand it — rewrite it.

---

## Writing Guardrails

Apply these rules to **every single word** on the page:

* Use the **simplest word possible**, always
* **No big words. No jargon. No formal language.**
* **Short sentences only** — max 10 to 12 words per sentence
* Write for **busy, impatient people** — get to the point fast
* If someone can skim it in 30 seconds and understand it — it is good
* **Read it out loud** before publishing. If it sounds like a form or a government letter — rewrite it

### Word swap examples

| Instead of this | Write this |
| :--- | :--- |
| Financial assistance | Money support |
| Employment opportunity | Job opening |
| Submit an application | Apply now |
| Valid identification document | Your ID card |
| Proof of residence | Proof of where you live |
| Verification process | We check your details |
| Eligibility criteria | Who can apply |
| Remuneration | Pay |
| Commence | Start |
| Required documents | What you need to bring |

---

## Page Sections

Design a **mobile-first** service information page with these 7 sections:

---

### 1. Page Header
* Short, plain service title (e.g. *"Get money support"* — not *"Financial Assistance Programme"*)
* Optional: quick read time and view count badges
* Dark green background (`#1a4a2e`) with amber/gold (`#e6a817`) accent

### 2. What is this? *(About this service)*
* 2 to 3 short sentences **maximum**
* Answer: What is it? Who is it for? What do they get?
* No fluff. No formal language.
* Use a left gold border accent on the card

### 3. What you get *(Benefits)*
* 3 to 5 bullet points with checkmark icons
* Start each line with an action word: *Get… Save… Access…*
* Each line should be one short sentence only

### 4. What you need to bring *(Required documents)*
* Numbered list (1, 2, 3…)
* Use everyday names — *"Your ID card"* not *"Valid Government-issued Identification"*
* 3 to 6 items maximum
* Place inside a soft card with a left gold border

### 5. How to do it *(Steps to apply)*
* 3 to 5 numbered steps
* Each step has: **Bold short title** + one sentence description
* No step description should be longer than 12 words
* Use circular amber number badges

### 6. People also ask *(FAQ)*
* Accordion-style — tap to expand each answer
* Write questions the way a **real person** would ask them (e.g. *"Do I pay anything?"*)
* Each answer = **1 sentence only**
* 3 FAQs minimum

### 7. Action Button *(CTA)*
* Full width, bottom of page
* Bold, plain action text — e.g. *"Apply now"*, *"Get started"*
* Amber/gold background (`#e6a817`), white text, rounded corners

## Quick Checklist Before Publishing

- [ ] Every sentence is under 12 words
- [ ] No jargon or formal language anywhere
- [ ] All section titles use plain everyday words
- [ ] FAQs are written how a real person would ask
- [ ] CTA button text tells people exactly what happens next
- [ ] Read the whole page out loud — if it sounds stiff, rewrite it
